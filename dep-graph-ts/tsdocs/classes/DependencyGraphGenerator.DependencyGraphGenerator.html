Class DependencyGraphGenerator

Description

Main orchestrator for analyzing a TypeScript/JavaScript project. It identifies modules, processes their Abstract Syntax Trees (ASTs) to extract dependencies, exports, and class/interface structures. Based on the provided options, it generates various outputs:

  • DOT files for different graph visualizations (package, relations, exports, class diagrams).
  • SVG files rendered from the DOT files using Graphviz.
  • JSON files containing the raw extracted data.
  • HTML reports summarizing the project structure and linking to diagrams.
  • Source code documentation using TypeDoc for TypeScript and JSDoc for JavaScript.

Property

The absolute path to the base file or directory being analyzed.

Property

The configuration options guiding the generation process.

Property

The absolute path to the directory where all outputs will be saved.
