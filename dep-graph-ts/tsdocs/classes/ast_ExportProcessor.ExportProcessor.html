Class ExportProcessor

Description

Processes Abstract Syntax Trees (ASTs) to extract ES module export information. It handles various export forms including:

  • export default ...
  • export const/let/var ... (with support for destructuring)
  • export function/class/interface/type/enum ...
  • export { name1, name2 as alias2 } from './module'; (re-exports)
  • export * from './module'; (star re-exports)
  • export * as ns from './module'; (namespace re-exports) Note: This processor is designed for ES module syntax and does not handle CommonJS-style exports (e.g., module.exports).
